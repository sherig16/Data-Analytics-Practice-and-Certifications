👋 Welcome to my repository containing a collection of data analysis practice questions and certification exams.

Each practice is organized in its own folder and typically includes:

- Data files (raw and/or processed, where applicable)  
- Analysis files (e.g., Excel workbooks, SQL scripts)  
- Visual outputs or screenshots for key results
